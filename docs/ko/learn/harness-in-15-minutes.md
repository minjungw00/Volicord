# 15분 만에 보는 하네스

## 짧아진 문서

이 문서는 줄였습니다. 처음 읽는 사람을 위한 둘러보기는 이제 [하나의 작업으로 보는 하네스](one-task.md)에 있습니다. 예전의 짧은 시나리오들을 하나의 사용자 작업 흐름으로 합쳤습니다.

오래된 링크를 위해 이 경로는 남겨 둡니다. 하지만 별도의 필수 학습 문서로 읽을 필요는 없습니다.

## 빠른 경로

대신 아래 순서로 읽습니다.

1. [개요](overview.md): 하네스가 무엇이고 왜 필요한지.
2. [하나의 작업](one-task.md): 작업 하나에서 어떻게 느껴지는지.
3. [핵심 개념](concepts.md): 최소 어휘.

짧은 생각은 그대로입니다. 사용자는 "이 계획을 구현할 수 있을 만큼 구체화해줘"처럼 평소 말로 요청할 수 있습니다. 에이전트는 필요한 것만 구체화하고 목표, 비목표, 완료 기준, 모르는 점, 사용자 소유 판단, 다음 안전한 구현 조각을 이름 붙여야 합니다. 하네스는 범위, 사용자 소유 판단, 증거, 확인, 최종 수락, 닫기 가능 여부, 잔여 위험이 대화에만 남지 않게 합니다.

이 문서는 API 안내서가 아니며, 이 문서 저장소에 실행 중인 하네스 서버가 있다는 뜻도 아닙니다.
