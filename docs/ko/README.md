# 하네스 문서

하네스 한국어 문서의 진입점입니다. 목적에 맞는 경로를 고른 뒤, 자세한 참조 내용은 연결된 담당 문서에서 확인합니다.

읽기 경로:

- 처음 읽는 사용자: [시작하기](start.md)를 읽은 뒤 [사용자 가이드](use/user-guide.md)로 이동합니다.
- 실제 흐름을 확인하는 사용자: [사용자 가이드](use/user-guide.md), [판단 예시](use/judgment-examples.md), [접점별 레시피](use/surface-recipes.md)를 봅니다.
- 에이전트와 운영자: [에이전트 가이드](use/agent-guide.md)를 사용하고, 저장소 편집 규칙은 [AGENTS.md](../../AGENTS.md)에서 확인합니다.
- 구현 담당자와 검토자: [구현 가이드](build/implementation-guide.md)를 읽은 뒤 [참조 색인](reference/README.md)에서 담당 문서를 찾습니다.
- 문서 유지보수자: [작성 가이드](maintain/authoring-guide.md), [번역 가이드](maintain/translation-guide.md), [점검 색인](maintain/checks.md), [doc-index.yaml](../doc-index.yaml), [용어 지도](../terminology-map.yaml)를 사용합니다.

정확한 담당 문서 경로는 [doc-index.yaml](../doc-index.yaml)을 기준으로 삼습니다. 사람이 읽기 쉬운 참조 경로는 [참조 색인](reference/README.md)을 사용합니다.
