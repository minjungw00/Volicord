# Harness 문서

이 문서는 Harness 한국어 문서의 진입점입니다. 읽을 경로를 고르는 데 사용하며, 자세한 참조 내용은 각 담당 문서가 정의합니다.

## 읽기 경로

- 처음 읽는 사용자: [시작하기](start.md)를 읽은 뒤 [사용자 가이드](use/user-guide.md)로 이동합니다.
- 작업 중인 사용자: [사용자 가이드](use/user-guide.md), [판단 예시](use/judgment-examples.md), [접점별 레시피](use/surface-recipes.md)를 봅니다.
- 에이전트 작성자나 운영자: [AGENTS.md](../../AGENTS.md), [에이전트 가이드](use/agent-guide.md), [에이전트 통합 참조](reference/agent-integration.md)를 사용합니다.
- 구현 담당자나 검토자: [구현 가이드](build/implementation-guide.md)를 읽은 뒤 [참조 색인](reference/README.md)에서 담당 문서를 찾습니다.
- 문서 유지보수자: [작성 가이드](maintain/authoring-guide.md), [번역 가이드](maintain/translation-guide.md), [점검 색인](maintain/checks.md), [doc-index.yaml](../doc-index.yaml), [용어 지도](../terminology-map.yaml)를 사용합니다.

## 담당 문서 찾기

- API, 스키마, 저장소, 보안, 범위, 상태 보기, 닫기 준비 상태의 담당 문서는 [참조 색인](reference/README.md)에서 찾습니다.
- 안정적인 `doc_id` 경로 메타데이터가 필요하면 [doc-index.yaml](../doc-index.yaml)을 사용합니다.
