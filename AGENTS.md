# Codex Working Rules

This repo is in pre-MVP Harness documentation redesign mode. Keep this file as a short docs-only compass.

- Do not implement the Harness server, runtime code, generated operational files, or product implementation yet.
- Documentation-only changes are allowed in this pre-MVP redesign mode.
- Work in `docs/en` first; mirror semantic documentation changes in `docs/ko` in the same batch.
- Maintain semantic parity between English and Korean docs, while allowing natural Korean headings and prose.
- Use small batches and report changed files.
- Do not create commits unless the user explicitly asks for commits.

## Docs-Authoring Override

Documentation edits are permitted in this pre-MVP redesign mode. This permission is not a Harness state transition, write authorization, evidence record, verification result, QA result, acceptance decision, or task close.
