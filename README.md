# Harness Project / 하네스 프로젝트

Harness is an agency-preserving local operating kernel for AI-assisted development. This repository currently holds the reviewed initial documentation baseline and is now in a reader-centered documentation redesign phase.

Harness는 AI 지원 개발에서 사용자의 판단권을 보존하는 로컬 운영 커널입니다. 이 저장소는 검토가 끝난 초기 문서 baseline을 바탕으로, 현재 독자 중심 문서 재설계 단계에 있습니다.

## Current Phase / 현재 단계

- The initial documentation baseline has been reviewed.
- The documentation set is now being redesigned around reader paths and practical entrypoints.
- Harness server and runtime implementation have not started.
- Implementation must start only after the redesigned documentation is accepted.

- 초기 문서 baseline 검토가 완료되었습니다.
- 문서 세트는 현재 독자 경로와 실용적인 진입점을 중심으로 재설계 중입니다.
- Harness server와 runtime 구현은 아직 시작하지 않았습니다.
- 구현은 재설계된 문서가 승인된 뒤에만 시작해야 합니다.
